Mythic Legends Card Set

The Mythic Legends set introduces 40 new cards featuring ancient creatures of myth and legend. This set includes the new "Mythic Rare" rarity level and special mechanics that affect card value.

Special Mechanics

  • Legendary: Extremely rare and powerful cards (50% value boost)
  • Ancient Power: Cards with historical significance (30% value boost)
  • Mythic Evolution: Cards that can evolve into more powerful forms (40% value boost)

Set Rotation

The Mythic Legends set is introduced in Year 2 and rotates out after Year 5. Card values increase before rotation, drop immediately after rotation, and then gradually increase again as they become collectibles.
